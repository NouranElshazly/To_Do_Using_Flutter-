# To-Do App

A simple to-do list app built with Flutter.

## Features

- Splash screen
- Authentication (login / sign up)
- Home screen with task management

## Getting Started

1. Install [Flutter](https://docs.flutter.dev/get-started/install).
2. Install dependencies:
   ```
   flutter pub get
   ```
3. Run the app:
   ```
   flutter run
   ```

## Tech Stack

- Flutter
- image_picker
- lottie
